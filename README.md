# GradeCalculator-Kotlin

Student Grade Calculator in Kotlin using OOP, higher-order functions, and collection operations.

## Description
This project reads student data from an Excel file, processes grades using OOP, displays results in a GUI, and exports to Excel and PDF.

## Features
- Read student data from .xlsx Excel file
- Calculate average grades per student
- Assign letter grades (A, B, C, D, F)
- Higher-order functions (map, filter, forEach)
- Graphical User Interface (Swing GUI)
- Export results to Excel and PDF

## Project Structure
- Milestone1_Abomo.kt — Core logic (OOP, classes, functions, Excel)
- MainGUI.kt — Graphical User Interface (Swing)
- build.gradle.kts — Project dependencies
- students.xlsx — Sample input Excel file

## How to Run
1. Open IntelliJ IDEA
2. Clone this repository
3. Sync Gradle
4. Run MainGUI.kt
5. Click "Charger Excel" to load students.xlsx
